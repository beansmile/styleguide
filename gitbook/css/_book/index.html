Beansmile CSS 编码规范

本文档是针对CSS设计的，但是使用Sass的时候，大部分规范也同样适用。 本文档的目的是希望团队的CSS代码风格一致，方便他人理解和维护。

注：规范中的遵守要求级别如下：

  • 标明 [强制] 的条目必须遵守
  • 标明 [建议] 的条目如无特殊情况，优先遵守
